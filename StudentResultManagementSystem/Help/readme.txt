The application requires Database by the name "studentresultsystem" and the 2 tables "student" and "result".
to setup the database or know more about the schema of the tables pls refer to database_and_table_creation_structures.sql file provided in the same folder
to setup pls run the commands in the .sql file using a user with sufficient permission
once database and table is setup you can find the admin page by loging in to the admin page 
with default username("admin") and password("passoword");
